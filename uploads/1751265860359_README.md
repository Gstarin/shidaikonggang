## Instalation 

1. Clone repository
```shell
git clone https://github.com/flatlogic/light-blue-vue-admin.git
```
2. Get in the project folder
```shell
cd light-blue-vue-admin
```
3. Install dependencies via yarn
```shell
yarn
```
